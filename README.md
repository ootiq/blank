# blank

Is the variable blank?

#### These functions are trivial, just code 'em yourself.

## Install

```
yarn add @ootiq/blank
```

## Usage

```javascript
const { isUndefined } = require("@ootiq/blank");

const a = null;

console.log(isUndefined(a));
// result:: => false
```
